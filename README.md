# DayMark: Year Events Calendar | Google™ - Public Website

Public-facing website for the DayMark: Year Events Calendar | Google™ browser extension, hosted on GitHub Pages.

## Purpose

This website serves as:
- **Application Homepage** for Google Cloud OAuth verification
- **Privacy Policy** page for compliance with Google API Services
- **Public information** about the extension

## Files

- `index.html` - Extension homepage
- `privacy.html` - Privacy policy for Google OAuth verification
- `img/logo.png` - Extension logo
